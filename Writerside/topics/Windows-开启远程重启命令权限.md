# Windows 开启远程重启命令权限

## 修改组策略

- Win+R 运行 `gpedit.msc` 打开**组策略编辑器**  
- 本地计算器策略
  - 计算机配置
    - Windows 设置
      - 安全设置
        - 用户权限分配
          - 从远程系统强制关机
            - 添加用户组 将你使用用户添加进去
            - 点击 应用 确定 

## 进行远程关机

登录远程设备

```Shell
net use \\ip地址\ipc$ '密码' /user:用户名

```
          
重启远程设备
```Shell
shutdown -r -m -f \\ip地址 -t  0
```
