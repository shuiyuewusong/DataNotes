# 网络磁盘 提示有害

Windows 安全中心  
打开这些文件可能会对你的计算机有害  
你的 Internet 安全设置阻止打开一个或多个文件. 是否仍要打开这些文件?

## 需要将网络磁盘的地址添加至受信任的站点

操作如下:

- 打开 Internet 选项
- 选择 Tab 安全
- 选择 受信任的站点
- 点击 站点(S)
- 取消 对该区域中的所有站点要求服务器验证(https:)(S) 的勾选
- 在 将该网络添加到区域(D): 中填写 网络磁盘的服务器 例: `192.168.4.4`  也可配置通配符: `192.168.*.*`
- 点击 添加(A)
- 点击 关闭(C)
- 点击 确定
